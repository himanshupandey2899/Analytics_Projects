# Shopping Mall Customer Segmentation Analysis

## Project Objective
Analyse customer behaviour and develop segmentation strategies
to support data-driven business decisions.

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- Data Visualization
- Business Analysis

## Project Workflow
1. Data Cleaning
2. Exploratory Analysis
3. Customer Segmentation
4. Interactive Dashboard Development
5. Business Recommendations

## Key Outcome
Developed an interactive Power BI dashboard enabling customer behaviour monitoring and strategic marketing decisions.
